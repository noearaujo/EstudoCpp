AND/&&/E
Operador lógico onde o valor de resposta da operação é verdadeiro 
se todas as variaveis de entrada forem verdadeiras(1).
Ex: 
0 and 0 == 0 (f)
0 && 1  == 0 (f)
1 and 0 == 0 (f)
1 && 1  == 1 (t)

OR/||/OU
Operador lógico onde o valor de resposta da operação é verdadeiro se
pelo menos uma das variaveis de entrada for verdadeira.
Ex:
0 or 0 == 0
0 || 1 == 1
1 or 0 == 1
1 || 1 == 1
